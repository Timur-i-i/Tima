#  Expense Tracker

## Информация об авторе
**Имя и фамилия** [Идрисов Тимур Ермекович]
**Дата создания:** 04.05.2026

## Описание

**Expense Tracker** — это простое и удобное приложение для учёта личных финансов. Оно позволяет записывать ежедневные расходы по категориям (еда, транспорт и т.д.), анализировать траты за любой период с помощью фильтрации и видеть общую сумму расходов. Все данные надёжно сохраняются в файл `expenses.json`, что позволяет вести учёт финансов постоянно.


## Требования

*   Язык программирования: *Python 3.x*.
*   Библиотека *GUI*: *Tkinter* (встроенная).
*   Формат данных: *JSON*.
*   Система контроля версий: *Git*.

## Использование приложения

### 1. Добавление нового расхода

Чтобы записать трату:
1.  Введите сумму в поле **«Сумма»** (например: `599.99`).
2.  Введите категорию в поле **«Категория»** (например: `Продукты` или `Бензин`).
3.  Выберите дату в виджете календаря **«Дата»**. По умолчанию там стоит сегодняшнее число.
4.  Нажмите кнопку **«➕ Добавить расход»**.
5.  Если данные введены верно, расход появится в таблице сверху.
6.  Поля ввода очистятся автоматически.
7.  Общая сумма расходов в нижней части экрана обновится.
8.  Данные будут сохранены в файл `expenses.json`.

### 2. Фильтрация и анализ трат

Чтобы посмотреть расходы за определённый период или по категории:
1.  Введите название категории в поле **«Категория»** (например: `Кафе`).
    *   *Приложение покажет только траты из этой категории.*
2.  Выберите даты начала и окончания периода в виджетах календаря под полями **«Дата начала»** и **«Дата окончания»**.
3.  Нажмите кнопку **«🔍 Применить фильтр»**.
4.  В таблице отобразятся только те расходы, которые соответствуют вашему запросу.
5.  Надпись «Сумма расходов» обновится и покажет итоговую сумму только для отфильтрованных данных.
6.  Чтобы вернуть полный список расходов — просто перезапустите приложение или очистите поля фильтра и нажмите кнопку снова.
