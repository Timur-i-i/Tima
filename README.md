## Описание

**Movie Library** — это кроссплатформенное приложение с графическим интерфейсом (*GUI*), написанное на языке *Python*. Оно предназначено для удобного хранения и управления вашей личной коллекцией фильмов.

**Ключевые возможности:**
*   **Добавление фильмов:** Вводите название (обязательное поле), жанр (обязательное поле), год выпуска и рейтинг.
*   **Валидация данных:** Приложение автоматически проверяет корректность ввода. Не позволяет ввести буквы в поле «Год» или рейтинг вне диапазона от 0 до 10.
*   **Отображение в таблице:** Все добавленные фильмы отображаются в удобной таблице с прокруткой.
*   **Фильтрация:** Быстро находите нужные фильмы по жанру или году выпуска.
*   **Сохранение данных:** Ваша коллекция сохраняется в файл `movies.json` и автоматически загружается при следующем запуске программы.
*   **Контроль версий:** Проект готов для работы с системой контроля версий *Git*.

## Требования

*   Язык программирования: *Python 3.x*.
*   Библиотека *GUI*: *Tkinter* (встроенная).
*   Формат данных: *JSON*.
*   Система контроля версий: *Git*.

## Использование приложения

### Добавление нового фильма:
1.  Введите название фильма (например: «Начало») в поле «Название».
2.	Укажите жанр (например: «Фантастика») в поле «Жанр».
3.	Введите год выпуска (например: 2010) в поле «Год выпуска».
4.	Оцените фильм по шкале от 0 до 10 (например: 8.8) в поле «Рейтинг».
5.	Нажмите кнопку «➕ Добавить фильм».
6.	Если данные введены верно, фильм появится в таблице сверху.
7.	Поля ввода автоматически очистятся для добавления следующего фильма.


### Фильтрация записей:
Если ваша коллекция стала большой и вы хотите быстро найти нужный фильм:
1.	Введите ключевое слово в поле «Жанр» (например: Комедия или Драма).
o	Приложение покажет только фильмы этого жанра.
2.	Или введите конкретный год в поле «Год» (например: 2023).
o	Приложение покажет только фильмы этого года.
3.	Нажмите кнопку «🔍 Применить фильтр».
4.	Чтобы вернуть полный список фильмов после фильтрации — просто очистите поля фильтра и нажмите кнопку еще раз или перезапустите приложение.
