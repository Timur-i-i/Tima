## Описание

Приложение «Movie Library» — это простая программа для хранения и фильтрации информации о фильмах. Позволяет добавлять фильмы с указанием названия, жанра, года выпуска и рейтинга. Данные сохраняются в формате JSON.


## Требования

Для работы приложения требуется: 
- Python 3.x.
- Tkinter (стандартная библиотека).
- JSON.
- Git.


## Ключевые возможности

1. Добавление фильмов: Вводите название, жанр, год выпуска и рейтинг.
2. Валидация данных: Приложение автоматически проверяет корректность ввода (например, не позволит ввести буквы в поле «Год» или рейтинг выше 10).
3. Отображение в таблице: Все добавленные фильмы отображаются в удобной таблице.
4. Фильтрация: Быстро находите нужные фильмы по жанру или году выпуска.
5. Сохранение данных: Ваша коллекция сохраняется в файл movies.json и загружается при следующем запуске.
6. Контроль версий: Проект готов для работы с системой контроля версий Git.


## Использование приложения

### Добавление нового фильма:
1.  Введите название фильма (например: «Начало») в поле «Название».
2.	Укажите жанр (например: «Фантастика») в поле «Жанр».
3.	Введите год выпуска (например: 2010) в поле «Год выпуска».
4.	Оцените фильм по шкале от 0 до 10 (например: 8.8) в поле «Рейтинг».
5.	Нажмите кнопку «➕ Добавить фильм».
6.	Если данные введены верно, фильм появится в таблице сверху.
7.	Поля ввода автоматически очистятся для добавления следующего фильма.

o	Приложение покажет только фильмы этого жанра.
2.	Или введите конкретный год в поле «Год» (например: 2023).
